# 错误情况测试用例

## 语法错误
1++2         # 连续操作符
1+           # 不完整表达式
+1           # 开头操作符
1+2*         # 不完整乘法
(1+2         # 缺少右括号
1+2)         # 缺少左括号
((1+2)       # 括号不匹配

## 除零错误
1/0          # 除零
2/0          # 除零
0/0          # 零除零

## 无效字符
1a+2         # 包含字母
1+2a         # 包含字母
1@2          # 无效操作符
1#2          # 无效操作符

## 空输入
             # 空字符串
             # 只有空格
tab          # 只有制表符

## 格式错误
1 + + 2      # 连续空格和操作符
1  +  2      # 过多空格
1+2+         # 结尾操作符
