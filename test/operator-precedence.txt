# 运算符优先级测试用例

## 加减乘除优先级
1+2*3        # 应该是 7
2*3+1        # 应该是 7
1+2*3+4      # 应该是 11
1*2+3*4      # 应该是 14

## 指数运算优先级
2**3*2       # 应该是 16
2*3**2       # 应该是 18
1+2**3*2     # 应该是 17
(1+2)**3     # 应该是 27

## 右结合性测试
2**3**2      # 应该是 512 (2**(3**2))
3**2**3      # 应该是 19683 (3**(2**3))

## 括号优先级
(1+2)*3      # 应该是 9
1+(2*3)      # 应该是 7
(2**3)**2    # 应该是 64
2**(3**2)    # 应该是 512

## 复杂优先级
1+2*3**2     # 应该是 19
(1+2)*3**2   # 应该是 27
1+(2*3)**2   # 应该是 37
